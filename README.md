
### Smart Express

Simple template Node.js Web App

##### Description

* HTTP Server: Express.js
* Template Engine: Nunjucks
* Style Engines: LESS
* Package Manager: yarn

##### Setup Node.js

    nodenv local

##### Deploy on IBM Cloud

    cf api https://api.eu-de.bluemix.net
    cf login --sso
    cf push

##### Initialize & Run project

    yarn install
    yarn browse 

On Mac use

    yarn browse-mac
